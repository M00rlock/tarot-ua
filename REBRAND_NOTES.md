# Tarot UA — Indigo Calm Rebrand v1

Зроблено перший крок у сторону тихого інді-ресурсу для саморефлексії.

## Що змінилось

- Додана нова головна сторінка `/`.
- Основна сесія винесена на `/session`.
- Додані заготовки сторінок `/journal` та `/library`.
- Додана спокійна навігація.
- Оновлена кольористика у сторону Indigo Calm.
- Фон став менш “магічним порталом” і більше схожим на нічний простір для роздумів.
- Hero у сесії переформульовано з “таро-розкладу” у “тиху сесію”.

## Новий напрям

Tarot UA тепер мислиться як особистий простір для саморефлексії:

- зупинитись;
- сформулювати думку;
- відкрити карти як символи;
- отримати м’яку рефлексію;
- зберегти власні нотатки.

## Перевірка

`npm --workspace apps/web run build` проходить успішно.
